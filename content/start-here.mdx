**What this is.** The single source of truth for the Arm Care Intelligence System (ACIS).
If a number or claim about ACIS lives anywhere else and disagrees with this site, this site
wins. Every page carries an owner and a last-reviewed date so you can tell what is current.

**What ACIS is, in one line.** A post-game arm-fatigue index for MLB pitchers. It reads a
pitcher's recent workload and performance-decay signals after each start and returns a
<Fact k="internal.scale" /> fatigue score. It is a fatigue meter, not an injury predictor, and not a real-time feed. {/* lint-language-disable-line: states the ban */}

**The one rule everyone follows.** ACIS is described as fatigue monitoring, workload
intelligence, arm health insights, or durability optimization.
Never as injury prediction or injury prevention. {/* lint-language-disable-line: states the ban */}
This holds in every document, deck, email, and conversation. The full language guide is in
[GTM Rails](/gtm-rails).
