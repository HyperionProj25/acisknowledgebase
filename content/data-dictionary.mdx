**Read this first.** This dictionary documents the fields that appear in the SP12 champion's
scored output, as defined in the validation context pack. It is the working reference, not
the complete feature manifest. The champion is documented as using <Fact k="internal.features" /> features; the fields
below are the ones named in the scored output and are fewer than <Fact k="internal.features" />. The full manifest lives
in the validation spreadsheets and model code, and exporting it is what makes this dictionary
authoritative. Until then, treat this as the confirmed subset.

## Champion output fields

These four are what the model produces per appearance.

| Field | Meaning |
| --- | --- |
| `risk_score` | The <Fact k="internal.scale" /> fatigue score. The primary output. |
| `probability` | Raw model probability behind the score, before tier mapping. |
| `risk_level` | Five-tier label. Note: the engine's text labels do not reliably match the published legend. Analyze by `risk_score`, not this field. |
| `recommended_action_code` | Coded guidance tier tied to the score band. |

## Champion input features (documented subset)

**Workload accumulation**

| Field | Meaning |
| --- | --- |
| `GamesPlayedSeason` | Appearances this season to date. |
| `SeasonPitches` | Pitches thrown this season to date. |
| `SeasonProgress` | Fraction of the season elapsed. Drives the calendar artifact; scoring is valid only through 0.65. |
| `AcuteLoad` | Short-window workload (recent pitches, fatigue-weighted). |
| `ChronicLoad` | Long-window baseline workload. |
| `ACWR` | Acute-to-chronic workload ratio. Needs about four starts of chronic baseline, which sets the 5th-start floor. |
| `WorkloadRisk` | Workload-derived risk component. |
| `FatiguePressureScore` | Composite fatigue-pressure signal. |
| `RestDays` | Days of rest before this appearance. |

**Performance decay**

| Field | Meaning |
| --- | --- |
| `FBAvgVelo` | Average fastball velocity for the appearance. |
| `VeloSlope` | Trend in fastball velocity across recent starts. A declining slope is a core fatigue proxy. |
| `SpinSlope` | Trend in spin rate across recent starts. |

**Injury history**

| Field | Meaning |
| --- | --- |
| `PriorArmILCount` | Count of prior arm-related IL stints. |
| `HadTommyJohn` | Prior Tommy John surgery flag. |

## Explainability output

| Field | Meaning |
| --- | --- |
| SHAP drivers (top 3) | The three features most responsible for this appearance's score. Commonly <Fact k="internal.topDrivers" />. |

## What makes this authoritative

Export the champion feature manifest (validation `.xlsx` or model code) and reconcile it
against the fields above. That closes the gap between this documented subset and the full
<Fact k="internal.features" />-feature set, and lets every feature carry a definition, source, and family.
